    该Dockerfile中包含了一些指令，其中：

  • FROM：指定基础镜像，本例中使用Node.js v14。

  • WORKDIR：设置工作目录为/app，该目录用于保存项目文件。

  • COPY：将项目文件复制到容器中的/app目录下。

  • RUN：执行npm install和npm run build命令，安装依赖并构建Vue项目。

  • EXPOSE：暴露容器的80端口。

  • CMD：指定容器启动时运行的命令，本例中为npm run serve。
