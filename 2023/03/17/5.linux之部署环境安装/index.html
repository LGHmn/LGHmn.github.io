5.linux之部署环境安装

本教程以centOS7为主

环境检查

  • 1.ip addr指令查看ens33的ip地址等信息
  • 2.ping或者curl百度网址检查网络是否连接成功，确保虚拟机网络可用

    安装docker

  • 1.更新系统sudo yum update
  • 2.安装 Docker 软件包sudo yum install docker
  • 3.启动 Docker 服务sudo systemctl start docker
  • 4.确认 Docker 服务已启动sudo systemctl status docker,如果 Docker 服务已经启动，那么您应该会看到一条active (running)的消息。
  • 5.配置 Docker 开机自启sudo systemctl enable docker
  • 6.验证 Docker 是否正常工作sudo docker run hello-world

获取最新版方法

  • 1.添加 Docker 官方 yum 仓库的 GPG 密钥：sudo rpm --import https://download.docker.com/linux/centos/gpg
  • 2.添加 Docker 官方 yum 仓库:sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
  • 3.更新 yum 软件包索引：sudo yum update
  • 4.安装 Docker CE：sudo yum install docker-ce
  • 5.启动 Docker 服务

    拉取数据库（pgsql为例）

  • 1.在确保docker已成功安装的情况下，可以在docker拉取postgresql的镜像。
  • 2.复制根据官网提供的版本号在linux终端输入docker pull postgres:xxx

  • 3.运行pgsql，输入docker run --name <为容器命名> -h 5432:5432 -e POSTGRES_PASSWORD=<设置密码> -d <拉取镜像版本号>
  • 4.linux终端输入docker ps -a查看该容器的状态
  • 5.连接数据库，同步数据库数据






    到此应该可以正常连接数据库，可以正常访问数据

安装nginx

  • 1.在linux终端输入rpm地址rpm -Uvh http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm
  • 2.终端输入yum -y install nginx安装指令，等待安装完毕
  • 3.输入systemctl start nginx启动nginx、systemctl enable nginx设置开机自启、systemctl status nginx查看状态

  • 4.docker运行目标容器

  • 5.将vue前端打包的dist文件夹内容粘贴到/usr/share/nginx/html路径下

  • 6.访问默认端口即可访问项目
